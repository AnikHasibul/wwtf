# WWWTF (World Wide Web Forensic Tool)

Basic forensic script for hacking the world wide web.

## Install:

```sh
cd
git clone https://github.com/AnikHasibul/wwwtf
cd wwwtf
chmod +x wwwtf
./wwwtf google.com
```

## Usage:

```
$ ./wwwtf apple.com
```
